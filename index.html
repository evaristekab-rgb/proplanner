How to use Pro Planner

Overview

  • Select Month/Year/Week using the header controls.
  • Click any day in Month view to add a task due that day.
  • Use the List tab to search & filter; Week tab to focus.
  • Everything is stored locally in your browser.

Recurring

  • Choose Daily/Weekly/Monthly and set Repeat Until. Instances are generated on save.

Burndown

  • Tracks remaining hours per day across the selected month. Ideal line is linear to month end.

Install (PWA)

  • Click Install when the button appears (or use browser “Install app”).

Backup

  • Export/Import JSON; export CSV for Excel/Power BI.
